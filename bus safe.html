🔥 ربط Firebase خطوة بخطوة
أنشئ مشروع Firebase مجاني
افتح console.firebase.google.com واضغط "Add project"، أدخل اسم المشروع مثلاً bussafe-uni، أوقف Google Analytics، ثم اضغط Create.
فعّل Realtime Database
من القائمة الجانبية اختر Build → Realtime Database → Create Database، اختر أي منطقة، ثم اختر "Start in test mode" (للتجربة).
احصل على مفاتيح التطبيق
اضغط أيقونة ⚙️ بجانب "Project Overview" → Project settings → مرر للأسفل → قسم "Your apps" → اضغط </> → سجّل اسم التطبيق → انسخ الـ firebaseConfig.
الصق المفاتيح في الكود
افتح ملف index.html في أي محرر نصوص (VS Code مثلاً)، ابحث عن firebaseConfig في أعلى الصفحة، والصق قيمك بدلاً من النص التجريبي.
const firebaseConfig = { apiKey: "AIzaSy...", authDomain: "bussafe-uni.firebaseapp.com", databaseURL: "https://bussafe-uni-default-rtdb...", projectId: "bussafe-uni", ... };
ارفع الملف على GitHub + Vercel
أنشئ repository على github.com، ارفع ملف index.html، ثم اذهب لـ vercel.com واربطه بالـ repository — سيعطيك رابطاً مثل bussafe.vercel.app في 60 ثانية.
